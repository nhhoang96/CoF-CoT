Use: File ending with _sp.txt for your purposes (i.e. train_en_sp.txt, test_en_sp.txt, eval_en_sp.txt)

Data Format:
Utterance \t Flat Representation \t Domain \t Language

Number of intents = 60 intents
Number of slots = 55 slots
